## **StrawberryBASIC**
ver 0.01a3

![StrawberryBASIC](https://gitee.com/steven-yang-blue/strawberry-basic/raw/master/STRAWBERRY-BASIC.png)

原创BASIC方言和解释器

已经实现了基本表达式、赋值、If-ElseIf-Else语句、While循环、Do-Loop循环、For循环
Break语句、Continue语句、Sub定义过程


安装方法（Ubuntu）：

- make flex
- make bison
- make compile

运行脚本Test.basic（Ubuntu）：

- ./BASIC < Test.basic
